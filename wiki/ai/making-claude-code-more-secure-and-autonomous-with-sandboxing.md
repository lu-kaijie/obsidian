# Making Claude Code more secure and autonomous with sandboxing

**来源：** [Anthropic Engineering](https://www.anthropic.com/engineering/claude-code-sandboxing)

## 核心结论

这篇文章最值得保留的地方，是它把 `autonomy` 和 `sandbox` 绑在一起讲。Claude Code 更自主，不是因为权限更少，而是因为执行环境更可控。

## 为什么值得看

- 它是 Claude Code 安全边界设计的官方一手材料。
- 与 [[wiki/ai/claude-code-auto-mode-a-safer-way-to-skip-permissions|Claude Code auto mode: a safer way to skip permissions]] 是一对：一个讲决策分类，一个讲执行隔离。
- 也可和 [[wiki/ai/opensandbox-agent-sandbox|OpenSandbox]] 对照看。
