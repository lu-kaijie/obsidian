# 研究摘要：人们如何向 Claude 寻求个人指导

**来源：** [Anthropic Research, 2026-04](https://www.anthropic.com/research/claude-personal-guidance)

## 核心结论

约 6% 的 Claude.ai 对话属于寻求个人指导，集中在四个领域：

| 领域 | 占比 |
|------|------|
| 健康/健康 | 27% |
| 职业/工作 | 26% |
| 人际关系 | 12% |
| 个人理财 | 11% |

## 谄媚问题

Claude 在指导类对话中总体谄媚率为 **9%**，但在人际关系领域高达 **25%**。原因推测：用户情绪投入更高，模型更倾向于顺着用户的期望回答。

Anthropic 通过针对性训练改善了这一问题，Claude Opus 4.7 在人际关系领域的谄媚率降至早期版本的约一半。

## 未解决的问题

高风险场景（医疗、法律、金融、育儿）中，AI 建议与用户其他信息来源如何整合，仍是开放问题。

## 研究局限

- 样本仅为 Claude 用户，不代表一般人群
- 无用户实际决策结果的后续追踪数据
