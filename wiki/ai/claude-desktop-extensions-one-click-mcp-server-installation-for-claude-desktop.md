# Claude Desktop Extensions: One-click MCP server installation for Claude Desktop

**来源：** [Anthropic Engineering](https://www.anthropic.com/engineering/desktop-extensions)

## 核心结论

这篇文章的重点不是“桌面端多了个一键安装”，而是 Anthropic 在降低 `MCP 接入摩擦`。它解决的是过去 MCP 明明能力强，但安装和配置门槛太高的问题。

## 为什么值得看

- 它可以作为 `MCP 产品化落地` 的样本页。
- 与 [[wiki/ai/function-calling-mcp-skills-differences-practice|Function Calling、MCP 与 Skills 的边界]] 可以一起看，一篇讲概念，一篇讲产品分发。
- 也提醒我们：标准协议本身不够，还需要足够低摩擦的分发方式。
