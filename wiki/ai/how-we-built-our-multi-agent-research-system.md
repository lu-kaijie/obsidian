# How we built our multi-agent research system

**来源：** [Anthropic Engineering](https://www.anthropic.com/engineering/multi-agent-research-system)

## 核心结论

这篇文章提供了一个很难得的官方样本：Anthropic 自己如何做 `multi-agent research system`。它的价值在于把研究任务中的多 agent 收益、提示词工程、评测和生产可靠性放在同一个系统里讲。

## 为什么值得看

- 它不是泛讲多 agent，而是聚焦 research 场景。
- 对照 [[wiki/ai/chat-window-to-multi-agent-console-collaboration-shift|从聊天窗口到多 Agent 控制台]] 和 [[wiki/ai/building-a-c-compiler-with-a-team-of-parallel-claudes|Building a C compiler with a team of parallel Claudes]]，可以看到 Anthropic 在不同任务类型上的多 agent 思路。

## 阅读评级

🔴 建议深读 — 如果你关心多 agent 不是为写代码，而是为研究和搜索类任务服务，这篇很有参考价值。
