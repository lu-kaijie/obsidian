# Effective context engineering for AI agents

**来源：** [Anthropic Engineering](https://www.anthropic.com/engineering/effective-context-engineering-for-ai-agents)

## 核心结论

这篇文章可以看作 Anthropic 对 `context engineering` 的官方总论。最核心的判断是：prompt 只是上下文的一部分，真正影响 agent 能力的，是如何检索、组织、压缩和持续维护上下文。

## 为什么值得看

- 它适合作为 context engineering 主线的一手总纲。
- 和 [[wiki/ai/openclaw-prompt-context-harness-philosophy|OpenClaw 的 Prompt / Context / Harness 设计哲学]]、[[wiki/ai/claude-code-prompt-context-harness-practice|Claude Code 的 Prompt / Context / Harness 设计实践]] 可以互相对照。
- 对长任务、agentic search、context retrieval 都有直接指导价值。

## 阅读评级

🔴 建议深读 — 如果你觉得“上下文工程”概念很泛，这篇能把它重新落回工程问题。
